# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Our standards
- Be respectful and constructive.
- Assume good intent.
- No harassment, discrimination, or hate speech.

## Enforcement
Project maintainers may remove, edit, or reject contributions that do not align with this code of conduct.

## Reporting
If you experience or witness unacceptable behavior, please open a private report via GitHub Security Advisories or contact the maintainers.
